When running `generate_next_steps.py`, paste the prompt output directly into this Claude Code conversation and respond with JSON here. Do NOT launch a separate Claude.ai tab or spawn an agent to call the API.

**Why:** The script is interactive (prints prompt, reads JSON from stdin). Claude Code IS Claude — just respond to the prompt inline, then feed the JSON back via stdin or write it directly to the JSONL.

**How to apply:** Any time the user says "run generate_next_steps" or "regenerate career cards," read the prompt files from `/tmp/prompt_<code>.txt` (already generated by `--print-prompt`) and produce the JSON directly in this conversation, then write to `data/output/occupation_cards.jsonl` via `append_career_page` logic or direct write.
